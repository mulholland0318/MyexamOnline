# MyexamOnline
在线考试系统有三个角色：管理员，教师和学生，实现试题管理，在线考试，自动评分和统计情况等功能<br>
该系统主要是利用DAO框架进行编写，开发涉及的主要技术有Java、HTML、CSS和JavaScript等<br><br>
